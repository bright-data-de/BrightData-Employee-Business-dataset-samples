## Datenpunkte

| Feldname | Beschreibung |
|------------|-------------|
| `url` | Die URL des Dataset-Eintrags. |
| `profile_url` | LinkedIn-Profil-URL der Person. |
| `linkedin_num_id` | Eindeutige numerische Kennung für das LinkedIn-Profil. |
| `avatar` | URL des Profilbilds, das mit dem LinkedIn-Konto verknüpft ist. |
| `profile_name` | Vollständiger Name der Person, wie in ihrem LinkedIn-Profil angegeben. |
| `certifications` | Details zu den von der Person erworbenen Zertifizierungen. |
| `profile_location` | Aktueller Standort der Person gemäß ihrem LinkedIn-Profil. |
| `profile_connections` | Ungefähre Anzahl der Verbindungen, die die Person auf LinkedIn hat. |
| `profile_country_code` | ISO-Code, der das Land des Standorts der Person angibt. |
| `profile_education_full` | Detaillierter Bildungsverlauf der Person. |
| `profile_last_education` | Jüngster Bildungsabschluss der Person. |
| `profile_experience_full` | Umfassende Details zur Berufserfahrung der Person. |
| `profile_last_experience` | Neueste Position und Arbeitgeber der Person. |
| `profile_followers` | Anzahl der Follower, die die Person auf LinkedIn hat. |
| `profile_linkedin_id` | LinkedIn-ID-String der Person. |
| `profile_current_position` | Aktuelle berufliche Position der Person. |
| `profile_current_title` | Aktuelle Berufsbezeichnung der Person. |
| `profile_activity` | Jüngste Aktivitäten der Person auf LinkedIn. |
| `profile_posts` | Von der Person auf LinkedIn veröffentlichte Beiträge. |
| `profile_about` | Zusammenfassungsabschnitt des LinkedIn-Profils der Person. |
| `profile_courses` | Von der Person absolvierte Kurse. |
| `profile_volunteer_experience` | Erfahrung der Person in ehrenamtlicher Arbeit. |
| `profile_languages` | Von der Person gesprochene Sprachen. |
| `profile_publications` | Von der Person verfasste Publikationen. |
| `profile_recommendations` | Empfehlungen, die die Person erhalten oder gegeben hat. |
| `profile_recommendations_count` | Gesamtzahl der Empfehlungen, die mit der Person verknüpft sind. |
| `profile_organizations` | Organisationen, denen die Person angehört oder angehört hat. |
| `profile_projects` | Von der Person durchgeführte Projekte. |
| `company_name` | Name des Unternehmens, bei dem die Person derzeit oder zuvor beschäftigt war. |
| `company_id` | Eindeutige Kennung für das Unternehmen innerhalb des Datasets. |
| `company_linkedin_url` | LinkedIn-URL des Unternehmens. |
| `company_size` | Größe des Unternehmens nach Mitarbeiterzahl. |
| `company_country_code` | ISO-Code, der das Land des Unternehmenshauptsitzes angibt. |
| `company_description` | Beschreibung der Geschäftstätigkeiten des Unternehmens. |
| `company_other_employees` | Anzahl anderer Mitarbeiter desselben Unternehmens, die auf LinkedIn gelistet sind. |
| `employees_in_linkedin` | Gesamtzahl der Mitarbeiter des Unternehmens, die auf LinkedIn gelistet sind. |
| `company_linkedin_followers` | Anzahl der Follower, die das Unternehmen auf LinkedIn hat. |
| `company_locations` | Liste der Unternehmensstandorte. |
| `company_founded_year` | Jahr der Unternehmensgründung. |
| `company_headquarters` | Hauptsitz des Unternehmens. |
| `company_categories` | Branchenkategorien, in denen das Unternehmen tätig ist. |
| `company_logo` | URL des Unternehmenslogos. |
| `company_slogan` | Slogan oder Claim des Unternehmens. |
| `company_specialties` | Fachgebiete und Spezialisierungen des Unternehmens. |
| `company_updates` | Jüngste Updates oder Neuigkeiten über das Unternehmen. |
| `company_website` | Offizielle Website des Unternehmens. |
| `company_type` | Unternehmensart, z. B. börsennotiert, privat, gemeinnützig. |
| `company_clean_domain` | Bereinigter Domainname der Unternehmenswebsite. |
| `company_revenue_usd` | Ungefährer Umsatz des Unternehmens in USD. |
| `company_total_funding` | Gesamtfinanzierung, die das Unternehmen erhalten hat. |
| `company_total_employees` | Gesamtzahl der Mitarbeiter, die im Unternehmen arbeiten. |
| `company_stock_symbol` | Börsenkürzel des Unternehmens, falls börsennotiert. |
| `company_is_non_profit` | Kennzeichnung, ob das Unternehmen eine Non-Profit-Organisation ist. |
| `company_parent_company` | Name der Muttergesellschaft, falls zutreffend. |
| `profile_bio_links` | Links, die im Biografie-Abschnitt des Profils der Person enthalten sind. |


## Anwendungsfälle

### 1. Human-Resources-Analytics
Verbessern Sie HR-Strategien, indem Sie Ausbildung, Erfahrung und Zertifizierungen von Mitarbeitern analysieren, um Kompetenzlücken und Schulungsbedarf zu identifizieren.

### 2. Marktforschung
Führen Sie Marktforschung durch, indem Sie Mitarbeiterprofile branchenübergreifend untersuchen, und helfen Sie Unternehmen so, Wettbewerbsdynamiken und die Verteilung der Belegschaft zu verstehen.

### 3. Geschäftsentwicklung
Nutzen Sie umfassende Mitarbeiter- und Unternehmensdaten, um Partnerschaftsmöglichkeiten aufzudecken und Maßnahmen zur Geschäftsexpansion strategisch zu planen.
